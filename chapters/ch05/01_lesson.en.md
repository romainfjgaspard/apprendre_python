# 📋 Chapter 5 — Lists

A **list** stores multiple values in order — like a team of Pokémon!

**What you will learn:**
- Create a list and access items by index
- Add and remove items
- Loop through a list
- Sort a list
- List comprehensions (create lists in one line!)

> In Scratch you have **"lists"** blocks — Python lists are very similar but more powerful.
