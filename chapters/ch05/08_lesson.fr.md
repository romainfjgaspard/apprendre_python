## 📖 Leçon 4 — Compréhensions de liste (la magie en une ligne !)

Crée une nouvelle liste en transformant ou filtrant une autre liste.

```python
# Ancienne façon :
result = []
for x in items:
    result.append(x * 2)

# Compréhension de liste :
result = [x * 2 for x in items]
```
