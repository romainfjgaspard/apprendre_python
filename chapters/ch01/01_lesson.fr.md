Bienvenue dans **Python Quest** ! Tu vas apprendre Python en construisant un jeu Pokémon.

**Ce que tu vas apprendre :**
- `print()` — afficher du texte à l'écran
- **Variables** — stocker des données (comme les variables Scratch 🟧)
- **f-strings** — mettre des variables dans du texte

**Comment jouer :**
1. Lis chaque cellule de cours 📖
2. Clique sur **▶ Exécuter** pour lancer les cellules de code
3. Remplis l'exercice ✏️ — remplace `???` par ton code
4. Clique sur **✅ Vérifier** en bas pour vérifier tes réponses

> 💡 Clique sur **▶ Exécuter** à côté d'une cellule de code pour l'exécuter
