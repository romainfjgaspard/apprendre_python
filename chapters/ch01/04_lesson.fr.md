## 📖 Leçon 2 — Variables : Stocke tes données

Une **variable** stocke une valeur — comme une variable Scratch 🟧.

```python
name = "Pikachu"    # texte → besoin de guillemets
hp = 35             # nombre → pas de guillemets
level = 5           # nombre → pas de guillemets
```

| Règle | Exemple |
|-------|---------|
| Texte → **guillemets** | `"Pikachu"` |
| Nombre entier → **pas de guillemets** | `35` |
| Décimal → **point, pas de guillemets** | `3.14` |
| Vrai/Faux → **majuscule, pas de guillemets** | `True` |
