## 📖 Lesson 2 — Variables: Store your data

A **variable** stores a value — like a Scratch variable 🟧.

```python
name = "Pikachu"    # text → needs quotes
hp = 35             # number → no quotes
level = 5           # number → no quotes
```

| Rule | Example |
|------|--------|
| Text → **quotes** | `"Pikachu"` |
| Whole number → **no quotes** | `35` |
| Decimal → **dot, no quotes** | `3.14` |
| True/False → **capital, no quotes** | `True` |
