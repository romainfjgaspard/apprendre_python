# 🔄 Chapitre 4b — La Boucle while

La boucle `while` répète du code **tant qu'une condition est True** — comme le bloc **"répéter jusqu'à"** de Scratch (mais inversé : while = tant que).

**Ce que tu vas apprendre :**
- La boucle `while` — répéter jusqu'à ce qu'une condition devienne False
- `break` — sortir d'une boucle en avance quand une condition est remplie
