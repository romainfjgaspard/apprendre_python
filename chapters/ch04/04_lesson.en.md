## 📖 Lesson 2 — Count with `for`: a list of numbers

You can also loop over a list of numbers to count steps.

```python
for i in [0, 1, 2, 3, 4]:
    print(f"Step {i}")
```

> This is like Scratch's **repeat (5)** block, but you can see each number.
> The variable `i` takes each value in the list, one by one.
