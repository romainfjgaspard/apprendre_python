## 📖 Leçon 1 — Les 4 types de base

| Type | Ce que c'est | Exemple |
|------|-------------|---------|
| `int` | Nombre entier | `hp = 35` |
| `float` | Nombre décimal | `speed = 3.14` |
| `str` | Texte (chaîne) | `name = "Pikachu"` |
| `bool` | Vrai ou Faux | `alive = True` |
