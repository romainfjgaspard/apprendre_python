# 🔢 Chapter 2 — Types & Math

Every value in Python has a **type**. The type tells Python what you can
do with the value.

**What you will learn:**
- The 4 basic types: `int`, `float`, `str`, `bool`
- Math operators: `+  -  *  /  //  %  **`
- Comparisons: `==  !=  <  >  <=  >=`
- How to calculate Pokémon battle damage!

> In Scratch every number is the same.
> In Python whole numbers (`int`) and decimals (`float`) are different types.
