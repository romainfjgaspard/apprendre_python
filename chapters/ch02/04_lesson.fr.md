## 📖 Leçon 2 — Opérateurs mathématiques

| Opérateur | Ce qu'il fait | Exemple | Résultat |
|-----------|--------------|---------|---------|
| `+` | Addition | `10 + 3` | `13` |
| `-` | Soustraction | `10 - 3` | `7` |
| `*` | Multiplication | `10 * 3` | `30` |
| `/` | Division | `10 / 3` | `3.333` |
| `//` | Division entière | `10 // 3` | `3` |
| `%` | Reste | `10 % 3` | `1` |
| `**` | Puissance | `2 ** 3` | `8` |

> 💡 `/` donne toujours un `float`. `//` donne toujours un `int` (arrondi vers le bas).
