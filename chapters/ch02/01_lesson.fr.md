# 🔢 Chapitre 2 — Types & Maths

Chaque valeur en Python a un **type**. Le type indique à Python ce que
tu peux faire avec la valeur.

**Ce que tu vas apprendre :**
- Les 4 types de base : `int`, `float`, `str`, `bool`
- Les opérateurs mathématiques : `+  -  *  /  //  %  **`
- Les comparaisons : `==  !=  <  >  <=  >=`
- Comment calculer les dégâts d'un combat Pokémon !

> Dans Scratch tous les nombres sont pareils.
> En Python les nombres entiers (`int`) et les décimaux (`float`) sont des types différents.
