## 📖 Leçon 3 — `and`, `or`, `not`

Combine plusieurs conditions :

| Mot-clé | Signification | Exemple |
|---------|---------------|---------|
| `and` | Les deux doivent être True | `hp > 0 and level >= 5` |
| `or` | Au moins une True | `type == "Fire" or type == "Water"` |
| `not` | Inverse True ↔ False | `not fainted` |
