## 📖 Leçon 1 — Ta première classe

```python
class Pokemon:
    def __init__(self, name, hp, attack):
        self.name   = name
        self.hp     = hp
        self.attack = attack
```
- `class` = mot-clé pour créer un plan
- `__init__` = s'exécute quand tu crées un nouveau Pokémon
- `self` = "cet objet" (Scratch : "ce sprite")
