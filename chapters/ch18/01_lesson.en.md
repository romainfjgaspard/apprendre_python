Welcome to **Season 3 — Birds Simulation**! 🐦

Before we build flying birds, we need to learn **classes** — Python's
way to group data + actions together.

**What you will learn:**
- `class` — a blueprint to create objects
- `__init__` — set up the object
- **Methods** — functions that belong to an object

> 💡 In Scratch, a **sprite** has variables (x, y, costume) and scripts.
> A Python **class** is similar: it has **attributes** (data) and **methods** (actions).
