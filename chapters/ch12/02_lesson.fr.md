## 📖 Leçon 1 — `import` un module

Un **module** est une boîte à outils. Pour l'utiliser, écris `import` en haut de ton fichier.
```python
import random
random.randint(1, 6)   # nombre aléatoire de 1 à 6 (comme un dé)
random.choice([...])   # choisit un élément dans une liste
random.shuffle(liste)  # mélange la liste sur place
```
