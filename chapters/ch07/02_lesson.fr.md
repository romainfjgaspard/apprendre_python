## 📖 Leçon 1 — `def` et `return`

```python
def nom_de_la_fonction(parametres):
    # code à l'intérieur de la fonction
    return resultat
```

- `def` = "je définis une fonction"
- Paramètres = les données que la fonction reçoit
- `return` = le résultat renvoyé
- L'indentation (4 espaces) = tout ce qui est à l'intérieur de la fonction
