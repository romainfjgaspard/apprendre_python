# 🧩 Chapitre 7 — Les Fonctions

Une **fonction** est un bloc de code réutilisable — comme un **bloc personnalisé** dans Scratch !

**Ce que tu vas apprendre :**
- `def` — définir une fonction
- Les paramètres et `return`
- Les valeurs par défaut pour les paramètres
- Les fonctions qui appellent d'autres fonctions
- Construire le système de combat Pokémon !
